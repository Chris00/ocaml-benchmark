Ocaml Benchmark is a small module to benchmark running times of code.  A typical use looks like:

        open Benchmark
        let res = throughputN 8 [("foo", f, 1000000);
                                 ("bar", g, 1000000);
                                 ("baz", h, 1000000) ] in
        print_newline();
        tabulate res

For a complete description of the functions, see the interface Benchmark.mli.
